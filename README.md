## 목업 웹서비스 기획 및 개발

### 1. Instagram Web Mockup :

  - **프로젝트명**: YouTube Web Mockup
  - **목적**: HTML/CSS를 활용해 유튜브 메인 페이지 UI를 모방하며 웹 프론트엔드의 기본 구조와 스타일링 기초를 익히기 위함
  - **기술 스택**: HTML5, CSS3
  - **주요 학습 목표**:
    - 시멘틱한 HTML 구조 이해
    - CSS 레이아웃 시스템 학습 (Flex, Grid)
    - 컴포넌트 분리 및 재사용 설계
    - 반응형 디자인 설계
  - **개발 기간** : 2025.06.11 ~ 2025.06.22
